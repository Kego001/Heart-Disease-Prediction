# Heart Disease Prediction using Decision Tree

## 📌 Project Overview

This project predicts whether a patient has heart disease using a Decision Tree Classifier. The model is trained on the Heart Disease dataset and demonstrates the complete machine learning workflow, from data preprocessing to model evaluation and saving the trained model.

---

## 📂 Dataset

- Dataset: Heart Disease Dataset
- Target Variable:
  - 0 → No Heart Disease
  - 1 → Heart Disease

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Joblib

---

## 📊 Workflow

- Import Libraries
- Load Dataset
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Selection
- Train-Test Split
- Decision Tree Model Training
- Model Evaluation
- Feature Importance Analysis
- Decision Tree Visualization
- Model Saving using Joblib

---

## 📈 Evaluation Metrics

- Accuracy Score
- Confusion Matrix
- Classification Report

The trained Decision Tree achieved excellent classification performance with high accuracy and strong precision, recall, and F1-score.
